这段时间做了一下针对IE 浏览器的问题，然后发现很多不兼容IE的地方

很多的框架也不支持IE 然后去找了找 发现一歌有趣的框架 ヽ(。_°)ノ avalon

迷你框架、使用简单、有较好的兼容性、注重性能、没有依赖

PS: 支持IE6+!!!!! 这才是重点

大概的看了一下这个框架 感觉和现在常用的VUE 和 AngularJS 有点儿像

首先从基本的开始吧

https://cdn.staticfile.org/avalon.js/2.2.4/avalon.js

那我们开始写程序员的标配hello-word

<div ms-controller="test">
  <input type="test" ms-duplex="@name"/>
  <p>hello,{{@name}}</p>
  <ul>
      <li ms-for="(item,key) in @array">{{item}}--{{key}}</li>
  </ul>
</div>

这个是简单的内容 但是这样是不行的 我们还需要相对应的数据 （个人觉得这就有点儿类似angular） 

<script>
  var vm = avalon.define({
      $id:"test",
    name:"world",
    array:[123,456,789]
  })
</script>

这样我们的数据就能显示在页面上了

这时候你可能会问了，上面这些标签里面的东西是什么啊

别急 这下我来和你说

1. ms-controller ---------------- 这条指令就是绑定对象 和 script 标签里面的$id 相对应
2. ms-duplex -------------------- 这条指令就是数据的双向绑定 类似 v-model(Vue)
3. {{}} ------------------------- 这条指令就是和vue的没有什么区别啦，绑定数据
4. ms-for ----------------------- 这条指令就是循环数组
5. @ ---------------------------- 我想这可能是这avalon的特色吧 @ + 'avalon.define'里面的字段 在加上{{就能把数据输出到页面上}}
完
